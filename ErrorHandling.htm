How to Handle JavaScript Errors

The try statement allows you to define a block of code to be tested for errors while it is being executed.

The catch statement allows you to define a block of code to be executed, if an error occurs in the try block.

The finally statement allows you to define a block of code to be executed, after try and catch, regardless of the result.

Types of errors

Reference Errors, Type Errors, Range Errors, URI Errors(uniform resource identifier), Syntax Errors, Eval Error (deprecated), Silent Errors .
